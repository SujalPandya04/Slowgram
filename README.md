# Slowgram

## Project Overview
Slowgram is an innovative social media platform designed to facilitate meaningful connections through slow-paced interactions. Users can share moments, thoughts, and experiences without the pressure of immediate responses or fast-paced timelines. The aim is to promote quality content and thoughtful engagement, encouraging users to take their time in sharing and interacting.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Deployment:** Heroku or AWS

## Setup Instructions
1. **Clone the repository:**  
   `git clone https://github.com/SujalPandya04/slowgram.git`

2. **Navigate into the project directory:**  
   `cd slowgram`

3. **Install dependencies for frontend and backend:**  
   - For the frontend, run:  
     `npm install` (in the frontend directory)  
   - For the backend, run:  
     `npm install` (in the backend directory)

4. **Set up environment variables:**  
   Create a `.env` file in the backend directory and add your configurations.

5. **Run the development server:**  
   - Start the backend:  
     `npm start` (in the backend directory)  
   - Start the frontend:  
     `npm start` (in the frontend directory)

6. **Open your browser:**  
   Visit `http://localhost:3000` to see the application in action.
